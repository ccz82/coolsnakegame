## Installation

Clone the project repository

```bash
  git clone https://github.com/ccz82/coolsnakegame.git
```

Go to the project directory

```bash
  cd coolsnakegame
```

Make sure pygame is installed

```bash
  pip install pygame
```

Start editing snake.py or to play it, run:

```bash
  python snake.py
```
